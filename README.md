# AFTERTRACE

AFTERTRACE is a compact vertical 3D hover racing game. Shift the AT-01 TWINBURN between three lanes, read designed hazard patterns, collect systems, and break through twenty city-and-factory stages.

- Keyboard: `A` / `D` or arrow keys
- Mobile: tap the left or right half of the screen
- Progress: clear stages in sequence and collect up to 30 stars
